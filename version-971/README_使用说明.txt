静态电影网站使用说明

1. 本网站已按上传的 data_2000.txt 实际解析结果生成，共 2000 部影片数据。
2. 首页、分类页、排行榜页、搜索页、详情页均为纯静态 HTML 文件。
3. 所有 HTML 页面都已插入百度统计脚本，脚本不会显示为页面文字。
4. 详情页播放器已绑定 m3u8 播放源，并通过 assets/js/player.js 调用 HLS 播放库。
5. 页面封面与 Hero 轮播按要求引用网站顶级目录下的 1.jpg 到 150.jpg。当前上传素材未包含这些图片文件，部署时请将 1.jpg、2.jpg ... 150.jpg 放在网站根目录，与 index.html 同级。
6. 直接上传整个解压后的目录到静态空间即可访问；如需本地测试，建议用本地服务器打开，例如 python -m http.server。
